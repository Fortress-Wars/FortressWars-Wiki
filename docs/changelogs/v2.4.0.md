### Pause Command Enhancements

- Added a "Paused" text overlay,
- Added a countdown when the game is being unpaused.
- /game pause now pauses:
  - Damage and healing processing.
  - Ability durations, cooldown timers, and refresh/regeneration timers.
  - Ability/background scheduled tasks (via PauseableTask/refactored base pause handling).
  - Potion effects and status effect ticking.
  - Respawn wave progression.
  - Assist tags, stun timers, hack-related tasks, and inventory pause state handling.
- Added discord hook to broadcast when a game pauses or unpauses.
