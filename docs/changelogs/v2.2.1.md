## Server Version

- Server updated from **1.21.8** to **1.21.11**

## Technical Changes

- Updated the syntax of the `/map` command.
- Added ability to set the map as an admin.
- Added configuration option to enable `map` and `gamerule` voting
- Moved Health Bar code into a standalone plugin.
- Added some status effects to health bars.
- Updated health bars to only show name through walls.
- Maps are now stored and loaded in the `maps/` directory instead of the root directory.

# Bug Fixes

- Fixed custom discord message not displaying.
- Fixed cobweb sound effect playing when earth walls are created.
