# v2.3.0

- [Server Version](#server-version)
- [Technical Changes](#technical-changes)
- [Bug Fixes](#bug-fixes)

## Server Version

- Server updated from **1.21.8** to **1.21.11**

## Technical Changes

- Updated the syntax of the `/map` command.
- Added ability to set the map as an admin.
- Added configuration option to enable `map` and `gamerule` voting
- Moved Health Bar code into a standalone plugin.
- Added some status effects to health bars.
- Updated health bars to only show name through walls.
- Maps are now stored and loaded in the `maps/` directory instead of the root directory.
- Player inventories are cleared when their kit is disabled.

# Bug Fixes

- Fixed custom discord message not displaying.
- Fixed cobweb sound effect playing when earth walls are created.
- Fixed extra space in miner's paxel death message.
- Fixed team griefing counting when breaking friendly kit blocks.
- Fixed ninja damage parry damage calculation.
  - The result is a less damage.
- Fixed snow golems dying in warm biomes.
- Fixed SNOW_GOLEM_SNOW_TRAIL_HEALING_PER_SECOND kit data property not accepting the double number type.
- Fixed issue where kit soldier could get stuck in the map after players teleported back to the lobby.
